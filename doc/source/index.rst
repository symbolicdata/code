|PACKAGE_NAME|
==============

The |PACKAGE_NAME| project is set out to develop concepts
and tools for testing Computer Algebra Software (CAS) and to collect
relevant data from different areas of Computer Algebra. Tools and data
are designed to be used both on a local site for special testing
purposes and to manage a central repository at www.symbolicdata.org.
SymbolicData tools and data are available for download under the terms
of the GNU Public License.

Note: This website is currently under construction. For further
details see `SymbolicData.org <http://symbolicdata.org>`_.
